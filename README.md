# football-cli

코딩하느라 바빠서 축구 못 볼 때 터미널에서 빠르게 스코어 확인 가능

# Installation

```shell
git clone https://github.com/wonkyum-kim/football-cli.git
cd football-cli
npm i
npm i -g
```

# Usage

```shell
$ score
? Select a League:  Bundesliga
? Select a team:  Bayern München
```
