# football-cli

코딩하느라 바빠서 축구 못 볼 때 터미널에서 빠르게 스코어 확인 가능

# Installation

```shell
git clone
cd football-cli
npm i -g
```

## Usage

```shell
$ score
? Select a League Premier League
? Select a team Newcastle United
[Game over]
Chelsea 3 - 2 Newcastle United
```

## League

- Premier League
- La Liga
- Bundesliga
- Serie A
- Ligue 1
